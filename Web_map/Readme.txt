Creating a simple Web Map Using "Folium" lib.

Folium is a third party lib written in python itself. What folium dose is it automatically converts the Python code used into HTML, JAVASCRIPT and CSS code so as to Produce the Web map which we can operate through browser.

For working :

    creat a new folder called mapping.
    unzip the the "world.json" in the mapping folder, also keep the "Volcaneos.txt" file here
    run the script.
